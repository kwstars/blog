### 光标移动操作
- ctrl + a：移动到最前面
- ctrl + e：移动到最后面
- ctrl + b：光标向前移动一格
- ctrl + f：光标向后移动一格

### 删除操作
- ctrl + q：删除所有内容
- ctrl + u：删除所有内容并复制
- ctrl + k：删除光标处到后面的所有内容
- ctrl + w：删除一个word
- ctrl + h：删除光标处一格

### 其他
- ctrl + t：光标处字母和前一个字母互换
- ctrl + p：显示上一次内容
- ctrl + r：搜索历史命令
- ctrl + y：粘帖`ctrl+u`的内容
- ctrl + o：相当与`enter`
- ctrl + d：有内容时相当于tab，无内容 exit
- ctrl + g：取消当前命令
- ctrl + j：执行命令
- ctrl +l：相当于`clear`命令

### esc组合键
- esc + . ：输入上一个命令的最后一个参数 
- esc + d：删除光标后的一个词
- esc + f：往右跳一个词
- esc + b：往左跳一个词
- esc + t：交换光标位置前的两个单词
